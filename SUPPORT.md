# Support

English | [한국어](SUPPORT.ko-KR.md)

Before opening an issue, check the README and try the basic troubleshooting steps.

## Installation Help

Use the `.rmskin` package if you want the easiest install path.

If you are downloading from GitHub and installing the skin yourself, use `.rmskin`.

Use the `.zip` package only for the built-in version manager / updater flow.

Before downloading, check the GitHub release notes for asset availability and SHA256 checksums. A Korea or Global variant can be temporarily unavailable; unavailable assets are listed in the release notes without checksum lines.

## Common Fixes

- Refresh Rainmeter.
- Restart Rainmeter.
- Check whether `Hide element` is enabled.
- Enable `Allow drag`, then reposition the element.
- Use `Reset skin position only` before resetting all settings.
- Open the log folder from the settings/debug area if helper actions fail.
- If an update or manual download is missing for your variant, check the release notes first; it may be temporarily unavailable.

## Where To Ask

- Confirmed bugs: GitHub Issues.
- Feature requests: GitHub Issues.
- General setup questions: use the support question issue form.
- Security reports: follow `SECURITY.md` and do not open a public issue.

## Useful Details To Include

- Skin version.
- Korea or Global package.
- Install method.
- Windows version.
- Rainmeter version.
- Screenshots or screen recording.
- Relevant logs.
